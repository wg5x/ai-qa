# 本地业务员 AI 助手 PRD（一期）

## 1. 背景

业务员在日常客户沟通中，经常会遇到以下问题：

- 客户提出不熟悉的问题，业务员不知道如何组织回复。
- 老客户返单时，需要快速查询历史报价、合同、型号、材质、价格等信息。
- 产品图片、视频、检测报告、包装对比等素材分散在本地文件夹，业务员很难在聊天过程中快速找到合适素材。
- 优秀业务员已经沉淀了一些谈单话术和成功聊天记录，但没有结构化整理，无法稳定复用。

一期目标是做一个本地网页应用，让业务员可以通过 AI 快速获得“回复思路 + 可直接发给客户的标准回复 + 相关历史报价 + 推荐图片/视频素材”。

## 2. 产品目标

### 2.1 一期目标

建立一个单个业务员本机使用的 AI 助手，支持：

1. 业务员用中文向 AI 提问，AI 用中文给出回复思路和标准回复。
2. 报价单、合同按固定 Excel 格式导入本地知识库。
3. AI 可以查询老客户历史型号、价格、材质、数量、合同信息，并与新订单需求进行比对。
4. 图片、视频、报告等素材以本地文件形式保存，并通过描述和标签被 AI 检索推荐。
5. 优秀聊天记录可导入，AI 总结成话术模板，经人工确认后才生效。

### 2.2 非目标

一期不做以下内容：

- 不做多人协作和共享知识库。
- 不做账号、权限、团队管理。
- 不做企业微信、飞书、WhatsApp 等聊天工具集成。
- 不做自动训练模型。
- 不允许 AI 未经人工确认就自动改变业务员话术风格。
- 不让 AI 一次性上传整个本地知识库到云端。
- 不优先支持任意格式 PDF、Word、图片报价单的自动识别，报价与合同先使用固定 Excel 模板导入。

## 3. 用户与使用场景

### 3.1 目标用户

单个外贸业务员，主要销售刹车片等汽车零部件产品。

### 3.2 核心场景

#### 场景一：客户问了业务员不会回答的问题

业务员输入：

> 客户问我们的刹车片有没有噪音，我应该怎么回复？

AI 输出：

- 回复思路：先解释产品经过测试，再说明低噪音来自材料配方和减震设计，最后建议客户先试样。
- 标准回复：可直接复制给客户的中文回复。
- 推荐素材：装车测试视频、检测报告、材料照片。

#### 场景二：老客户返单，需要查询历史价格

业务员输入：

> 查一下利比亚客户 Ahmed 之前买过 D1234 吗？当时什么价格？

AI 输出：

- 历史记录：客户、日期、型号、材质、数量、单价、币种、合同编号。
- 价格说明：历史价格是否可直接沿用。
- 风险提醒：如果材质、钢板厚度、包装、数量不同，需要重新核价。

#### 场景三：新订单型号与历史订单比对

业务员输入：

> Ahmed 这次要 D1234 和 D5678，帮我和他上次订单比一下。

AI 输出：

- 相同型号：D1234，显示上次配置和价格。
- 新增型号：D5678，提示需要核价或查报价库。
- 差异项：材质、钢板厚度、包装、数量、币种等变化。
- 建议回复：给客户的中文标准回复。

#### 场景四：客户关注包装或产品质量，需要推荐素材

业务员输入：

> 客户想看 HIQ 包装效果，帮我找素材并告诉我怎么说。

AI 输出：

- 回复思路：强调包装颜色、图案清晰度、纸张差异和工厂定制能力。
- 标准回复：中文客户回复。
- 推荐素材：HIQ 包装对比图片、开箱视频、纸张对比视频。

#### 场景五：导入优秀聊天记录，沉淀话术模板

业务员上传一段成功谈单聊天记录。

AI 输出：

- 适用场景：报价后客户嫌贵、客户问质量、客户犹豫是否下单等。
- 话术风格：例如直接、亲切、强调工厂身份、强调定制化。
- 话术模板：整理后的可复用模板。
- 人工确认按钮：确认后才进入知识库。

## 4. 一期功能需求

### 4.1 AI 问答

业务员可以在本地网页中输入问题，AI 需要基于本地知识库和当前问题生成答案。

#### 输出格式

AI 每次回答默认包含：

1. 回复思路
2. 标准回复
3. 参考依据
4. 推荐素材
5. 注意事项

#### 输出要求

- 全部使用中文。
- 标准回复必须像真实业务员说话，不能像 AI 说明书。
- 标准回复要能直接复制给客户。
- 不确定的信息必须明确提示“需要人工确认”。
- 内部成本、供应商底价、报价公式等敏感信息不能写入标准回复。

### 4.2 本地知识库

一期知识库只在业务员本机保存。

知识库内容包括：

- 谈单手册
- 常见问题与标准话术
- 报价单
- 合同记录
- 客户资料
- 图片素材
- 视频素材
- 素材描述和标签
- 已确认的话术模板

### 4.3 报价单与合同导入

报价单、合同统一通过固定 Excel 格式导入。

#### 报价单 Excel 字段

| 字段 | 说明 | 是否必填 |
| --- | --- | --- |
| 客户名称 | 客户或公司名称 | 是 |
| 国家/地区 | 客户所在市场 | 是 |
| 型号 | 刹车片型号或 OE 号 | 是 |
| 替换号码 | KD、SP、FMSI、OE 等替换号码 | 否 |
| 材质等级 | 普通半金属、A+、AA、AAA、陶瓷、碳纤维等 | 是 |
| 钢板厚度 | 4MM、5MM 或其他 | 否 |
| 是否带减震片 | 是/否 | 否 |
| 包装方式 | 客户品牌、我方品牌、仿牌、裸装等 | 是 |
| 数量 | 报价数量 | 是 |
| 单价 | 报价单价 | 是 |
| 币种 | RMB、USD 等 | 是 |
| 报价日期 | 报价时间 | 是 |
| 有效期 | 报价有效期限 | 否 |
| 备注 | 特殊要求 | 否 |

#### 合同 Excel 字段

| 字段 | 说明 | 是否必填 |
| --- | --- | --- |
| 合同编号 | 合同唯一编号 | 是 |
| 客户名称 | 客户或公司名称 | 是 |
| 国家/地区 | 客户所在市场 | 是 |
| 下单日期 | 合同日期 | 是 |
| 型号 | 订单型号 | 是 |
| 材质等级 | 实际订单材质 | 是 |
| 包装方式 | 实际订单包装 | 是 |
| 数量 | 订单数量 | 是 |
| 单价 | 合同单价 | 是 |
| 币种 | RMB、USD 等 | 是 |
| 交货期 | 约定交货时间 | 否 |
| 付款方式 | 定金、尾款等 | 否 |
| 备注 | 特殊要求 | 否 |

#### 导入要求

- 导入前检查必填字段。
- 同一客户、同一型号、同一日期的重复记录需要提示业务员确认。
- 导入后支持按客户、国家、型号、日期、材质、包装方式检索。
- 原始 Excel 文件保留在本地，系统生成结构化索引。

### 4.4 历史价格查询与新订单比对

AI 需要支持以下查询：

- 查询某客户历史购买过哪些型号。
- 查询某客户某型号历史价格。
- 查询某型号在不同客户、国家、材质下的价格差异。
- 将新订单型号与历史订单进行比对。
- 提醒业务员哪些字段会影响报价。

#### 比对维度

- 型号是否相同
- 替换号码是否相同
- 材质是否相同
- 钢板厚度是否相同
- 是否带减震片
- 包装方式是否相同
- 数量是否变化
- 币种是否变化
- 报价日期是否过久

### 4.5 图片与视频素材库

业务员可以把图片、视频、检测报告等素材放入本地素材库。

每个素材需要有结构化描述。

#### 素材字段

| 字段 | 说明 | 是否必填 |
| --- | --- | --- |
| 素材名称 | 文件显示名称 | 是 |
| 文件路径 | 本地文件路径 | 是 |
| 素材类型 | 图片、视频、报告、其他 | 是 |
| 产品类型 | 刹车片、包装、外箱、材料、检测报告等 | 是 |
| 适用场景 | 质量说明、包装说明、材料介绍、交货期说明等 | 是 |
| 相关品牌 | HIQ、ASIMCO、EMVP、客户品牌等 | 否 |
| 相关材质 | A+、AA、AAA、陶瓷、碳纤维等 | 否 |
| 中文描述 | 业务员可理解的素材说明 | 是 |
| 推荐话术 | 发送素材时搭配的说明 | 否 |
| 标签 | 多个关键词 | 否 |

#### 素材推荐规则

AI 根据业务员问题和客户场景推荐素材。

示例：

- 客户问质量：推荐检测报告、装车测试视频、材料照片。
- 客户问噪音：推荐装车实验视频、低噪音材料说明。
- 客户问包装：推荐包装对比图、开箱视频、纸张对比视频。
- 客户担心运输损坏：推荐外箱承重视频。
- 客户要了解材质：推荐六种材料实物照片和检测报告。

### 4.6 话术模板学习

一期不做自动训练模型，只做“导入优秀话术 → AI 总结 → 人工确认 → 生效”。

#### 流程

1. 业务员上传优秀聊天记录或手动粘贴对话。
2. AI 分析聊天记录。
3. AI 提炼适用场景、客户问题、业务员回复方式、谈判技巧、标准模板。
4. 业务员人工审核。
5. 审核通过后，话术模板进入知识库。
6. 后续 AI 问答可以引用该模板。

#### 人工确认要求

未确认的话术模板不能自动用于正式回答。

确认时业务员可以修改：

- 适用场景
- 标准回复
- 禁用词
- 推荐素材
- 注意事项

## 5. 页面需求

### 5.1 首页/问答页

主要功能：

- 输入业务问题
- 查看 AI 回复思路
- 查看标准回复
- 查看相关报价/合同记录
- 查看推荐素材
- 一键复制标准回复
- 打开本地图片/视频文件

### 5.2 知识库管理页

主要功能：

- 查看已导入资料
- 导入报价单 Excel
- 导入合同 Excel
- 导入谈单手册
- 搜索知识库内容
- 删除或更新本地资料

### 5.3 素材管理页

主要功能：

- 添加图片/视频/报告
- 编辑素材描述
- 编辑标签
- 设置适用场景
- 预览图片或视频

### 5.4 话术模板页

主要功能：

- 上传或粘贴优秀聊天记录
- 查看 AI 总结的话术模板
- 修改模板
- 确认生效
- 禁用不再适用的话术模板

## 6. AI 回答规则

### 6.1 默认回答结构

```text
回复思路：
说明应该如何回答客户，以及为什么这样回答。

标准回复：
给出可以直接复制给客户的中文回复。

参考依据：
列出引用了哪些知识库内容、报价记录、合同记录或话术模板。

推荐素材：
列出适合发送给客户的图片、视频或报告。

注意事项：
提醒哪些信息需要人工确认，哪些内部信息不能发给客户。
```

### 6.2 安全规则

- 内部报价公式、供应商成本、利润空间不能直接发给客户。
- 合同和报价信息只能在业务员本机检索。
- 云端 AI 只接收当前问题相关的必要片段。
- 涉及价格时，如果历史报价过久或配置不同，必须提醒重新核价。
- 遇到知识库没有依据的问题，AI 不能编造答案。

## 7. 数据与隐私要求

### 7.1 本地存储

以下数据默认保存在本地：

- 原始 Excel 文件
- 结构化报价数据
- 结构化合同数据
- 客户资料
- 图片和视频素材
- 聊天记录
- 话术模板

### 7.2 云端 AI 调用

允许调用云端 AI，但必须满足：

- 只上传与当前问题相关的知识片段。
- 不上传整个本地知识库。
- 不上传无关客户资料。
- 不上传无关合同和报价记录。
- 业务员可以查看 AI 引用了哪些本地资料。

## 8. 成功标准

一期完成后，应满足以下标准：

1. 业务员可以在本地网页中提问，并获得中文回复思路和中文标准回复。
2. 固定格式 Excel 报价单可以成功导入并被检索。
3. 固定格式 Excel 合同可以成功导入并被检索。
4. 可以查询某客户历史型号和历史价格。
5. 可以对比老订单与新订单型号、材质、包装、数量、价格差异。
6. AI 能根据问题推荐相关图片或视频素材。
7. 优秀聊天记录可以被 AI 总结成话术模板。
8. 话术模板必须人工确认后才会用于后续问答。
9. AI 不会把内部成本、报价公式、底价写进标准回复。

## 9. 一期验收用例

### 用例一：常见问题回复

输入：

> 客户问刹车片有没有噪音，怎么回复？

期望：

- AI 给出中文回复思路。
- AI 给出中文标准回复。
- AI 推荐相关测试视频或检测报告。

### 用例二：历史价格查询

输入：

> 查一下 Ahmed 之前 D1234 的价格。

期望：

- AI 返回历史合同或报价记录。
- AI 显示型号、材质、数量、单价、币种、日期。
- 如果没有记录，AI 明确说明未找到。

### 用例三：新老订单比对

输入：

> Ahmed 这次要 D1234 和 D5678，帮我和上次订单比一下。

期望：

- AI 区分历史型号和新增型号。
- AI 标出配置变化。
- AI 提醒需要重新核价的项目。

### 用例四：素材推荐

输入：

> 客户想看 HIQ 包装效果。

期望：

- AI 推荐 HIQ 包装相关图片或视频。
- AI 给出发送素材时搭配的中文话术。

### 用例五：话术模板确认

输入：

> 上传一段成功谈单聊天记录。

期望：

- AI 总结适用场景、话术风格、标准模板。
- 模板默认处于“待确认”状态。
- 业务员确认后，该模板才进入正式知识库。

## 10. 后续版本方向

以下内容不进入一期，但可以作为后续版本：

- 多业务员共享知识库。
- 权限管理和团队数据隔离。
- 飞书、企业微信、WhatsApp 等聊天工具集成。
- PDF、Word、图片报价单自动识别。
- 自动生成跟进提醒。
- 自动分析客户质量和成交概率。
- 自动生成报价单或合同草稿。
- 更细的素材效果统计，例如哪些图片或视频更容易促成回复。
